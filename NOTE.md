### 初始化项目
```vim
npm init @vitejs/app
```
### 依赖
```
npm install

npm install vue-router@next vuex@next
```

### 项目目录
```
├── src
│   ├── api            数据请求
│   ├── assets         静态资源
│   ├── components     组件
│   ├── pages          页面
│   ├── router         路由配置
│   ├── store          vuex数据
│   └── utils          工具函数
```
然后配置路由设置
